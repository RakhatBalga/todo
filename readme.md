# To-Do List App

A simple **To-Do List** application built with **HTML, CSS, and vanilla JavaScript**.  
The goal of this project is to practice **state-driven UI**, **DOM manipulation**, and **clean project structure** without using frameworks or localStorage.

---

## Features

- Add new tasks via input field
- Prevent empty tasks (`trim()` validation)
- Display tasks in a list
- Delete individual tasks
- Clean and responsive UI
- No external libraries
- No localStorage (state exists only in memory)

---

## Project Goals

This project was created to learn and practice:

- Working with user input
- Event handling (`click`)
- Managing application state with arrays
- Rendering UI from state (`render()` pattern)
- Creating and manipulating DOM elements
- Separating **logic (state)** from **presentation (DOM)**
